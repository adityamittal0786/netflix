# Netflix Homepage Design Reference

## Key Design Elements

### Color Scheme
- Primary: Black background (#000000 or very dark)
- Accent: Netflix Red (#E50914)
- Text: White for primary text
- Secondary: Gray for supporting text

### Layout Structure
1. **Header/Navigation**
   - Netflix logo (left)
   - Language selector (top right)
   - Sign In button (top right)

2. **Hero Section**
   - Large background image with movie posters collage
   - Dark gradient overlay for text readability
   - Main headline: "Unlimited films, series and more"
   - Subheading: Pricing info
   - Email input + "Get Started" CTA button (red)

3. **Trending Section**
   - "Trending now" title
   - Horizontal scrollable carousel of movie tiles
   - Each tile shows: poster image + title + ranking number
   - Numbered badges (1-10) in yellow/gold

4. **Features Section**
   - 4 feature cards with icons and descriptions
   - Dark background with white text
   - Simple, clean layout

5. **FAQ Section**
   - Accordion-style expandable questions
   - Dark background

6. **Footer**
   - Links organized in columns
   - Language selector
   - Copyright info

### Typography
- Bold, large headlines
- Clean, readable body text
- Sans-serif font family

### Interactive Elements
- Red "Get Started" button (primary CTA)
- Hover effects on movie tiles
- Carousel navigation arrows
- Accordion expand/collapse

### Visual Style
- Dark theme throughout
- Movie poster imagery as background
- Numbered ranking badges
- Minimal, modern aesthetic
- High contrast for readability
